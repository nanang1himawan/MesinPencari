Di sektor pertanian ada banyak petani jamur yang melakukan manual penanaman. Faktor lingkungan yang mempengaruhi pertumbuhan jamur tiram antara lain suhu dan kelembaban. Sistem ini dibuat untuk menjaga stabilitas suhu dan suhu kelembaban di rumah kaca jamur tiram dari jarak jauh. Komunikasi data jarak jauh pertukaran antara Raspberry-Pi dan Zelio menggunakan jaringan TCP / IP berhasil dilakukan di luar. Sistem berhasil menjalankan fitur di menu pengaturan yang dapat mengatur nilai suhu dan titik setel kelembaban, mengatur kondisi on / off dari aktuator, tampilan titik setel sistem lokal dan setel kondisi hidup / mati dari tombol aktifkan di GUI. Nilai suhu dan kelembaban, dan status aktuator dapat ditampilkan di menu pemantauan. Di menu logging, data disimpan dalam format csv. Penamaan file otomatis berdasarkan tanggal adalah berhasil dilakukan, file csv berhasil dibuka dan ditampilkan dalam plot grafis.
